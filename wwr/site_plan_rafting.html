Overview

Purpose

I am doing my senior project using JavaScript so this is my website idea. The purpose of the project is to create a simple but useful scheduling website so the CIT department can use it to schedule classes for each semester. The website is going to be able to save time for the secretaries and department chair.

Audience

The users are going to be secretaries and department chairs in the CIT department. If it is successful then other departments on campus can use the website as well.
